# 快捷键列表

| 操作           | 快捷键              |
| -------------- | ------------------- |
| 画布移动       | 按住鼠标中键移动    |
| 画布缩放       | Ctrl + 鼠标滚轮滚动 |
| 画点           | 鼠标左键            |
| 结束画点       | 鼠标右键            |
| 打开文件夹     | Ctrl + O            |
| 切分大图       | Ctrl + B            |
| 合并大图       | Ctrl + M            |
| 打开快捷键帮助 | Ctrl + H            |
| 保存           | Ctrl + S            |
| 切换上一张     | S                   |
| 切换下一张     | F                   |
| 图像全屏       | Ctrl + F            |
| 删除多边形     | Backspace           |
| 删除所有多边形 | Delete              |
